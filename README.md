# Prisoner's Dilemma Strategy Simulator

## Overview

This project simulates the classic Prisoner's Dilemma problem from Game Theory. Different strategies are compared to study cooperation, competition, and decision-making behavior in repeated interactions.

## Technologies and Concepts

- Python
- Artificial Intelligence
- Game Theory
- Strategic Decision Making
- Gradio

## Features

- Multiple player strategies
- Score comparison system
- Interactive simulation
- Strategy evaluation
- Decision analysis

## Strategies Implemented

- Always Cooperate
- Always Betray
- Tit for Tat
- Random Strategy

## Applications

- Artificial Intelligence
- Multi-Agent Systems
- Economics
- Negotiation Systems
- Strategic Analysis
